# amnesia

this is a fork of [*Amnesia: A Machine for Pigs*](https://store.steampowered.com/app/239200/Amnesia_A_Machine_for_Pigs/)
which aims to clean up and modernize the codebase, improve support for [*Amnesia: The Dark Descent*](https://store.steampowered.com/app/57300/Amnesia_The_Dark_Descent/) and its custom
stories, address bugs and compatibility issues, and more.

it's still a work-in-progress! but please feel free to look around..
